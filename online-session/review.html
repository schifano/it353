What data structure is used to model a shopping cart and what is the initial capacity of this data structure?

The data structure used to model the shopping cart is an array, and the initial capacity of the array is 3. Each item is an Item object with the properties of a name, price, and quantity. The Item objects are stored in the array.

When a shopping cart reaches its initial capacity, what happens?

When the shopping cart reaches its initial capacity, the method increaseSize() is called.

Explain in plain English how the data structure "stretches" to accommodate more items if/when it reaches the initial capacity.

When the increaseSize() method is called, it creates a temporary array using the current capacity size, doubles the capacity size variable, copies all the current array items into the temporary array, creates a new cart array with the new capacity size, and then copies items from the temporary array into the new array in order to create the new cart with double the capacity size. Briefly describe the purpose of using an "interface" in Java.

Interfaces allow an object to have multiple identities, or in otherwords, they can help simulate multiple inheritance.
