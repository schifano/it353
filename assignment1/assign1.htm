Short Answers

What does ARIA stand for? What problem of the Web does it address, and how does it do it?

ARIA is also known as WAI-ARIA, which stands for Accessible Rich Internet Applications Suite. ARIA is used to make web applications more accessible for the differently-abled individuals that surf the Internet using accessible technologies (ex: screen readers).

ARIA addresses Web accessibility issues by providing developers a framework that adds attributes to describe navigation for accessible technologies. It also helps to enable better keyboard functionality so that it is easier for people to move to different regions of content. Roles are used to describe the purpose and structure of different web parts and properties are used to communicate the different states they may be in. 1

What is search engine optimization (SEO) and why is it important?

Search engine optimization (SEO) is the process of attracting traffic from users from the natural search results on various search engines. Search engines such as Google, Bing, and Yahoo bring search results to users, but SEO optimizes those search results in order to determine the best fit for the user. When a website or web page is able to naturally rank higher in the search, their site is optimized to reach the largest audience possible.2

What is Cloud Computing and how is it different from the traditional computing involving data centers, etc.?

Cloud Computing is the ability to store and access data over the Internet instead of immediately through a computer hard drive. The metaphorical "cloud" represents the Internet where the data is stored. Cloud Computing is different from traditional computing in that any data can be accessed from any place virtually. It reduces the need for giant data centers and storage in residence, but it also introduces greater security risk than traditional computing.3

What is LAMP Stack? Give a benefit of using LAMP in a business.

LAMP is a software bundle that stands for Linux, Apache, MySQL, PHP (or Perl/Python). The LAMP stack allows for quick application development that is reliable and easy to integrate within an IT system. Businesses need to be prepared for business growth, and LAMP provides that.4

What is a Web (Application) Framework? Name 3 examples of this.

A web application framework is a foundation that helps developers build and design web applications. They provide functionality to web applications in order to save web developers time when building a full website.5

Three examples include:

  1. AngularJS
  2. EmberJS
  3. Ruby on Rails
